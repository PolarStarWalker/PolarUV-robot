# PolarUV-robot
This software is part of PolarUV project and runs on a raspberry pi 4 single board computer or similar.

# Dependencies
Software dependencies:
 - Debian 11 bullseye
 - gcc - 10.2
 - cmake >= 3.16
 - conan 1.47
 - all needed libraries automatically installing by conan

Hardware dependencies:
 - i2c
 - spi
